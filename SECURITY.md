# Security Policy

We take security seriously. Please do not disclose vulnerabilities publicly before we have a chance to investigate and release a fix.

## Reporting a Vulnerability

- Email: security@noverna.de
- Or open a private security advisory on GitHub ("Report a vulnerability" in the repo Security tab)

Provide:

- Affected version/commit
- Reproduction steps and impact
- Suggested remediation if known

## Scope

- This repository and official Noverna-maintained packages
- Third-party dependencies are in scope if they are bundled or required directly by this repo

## Handling

- We will acknowledge receipt within 72 hours
- We will investigate, prioritize, and work on a fix
- We will credit reporters if desired when publishing advisories/releases
